# Contributing to CLARA

Thank you for your interest in contributing to CLARA! We welcome contributions from the community to help improve clinical reasoning and patient care.

## How to Contribute

1. **Fork the repository** and create your branch from `main`.
2. **Install dependencies** with `npm install`.
3. **Create a .env file** (see README for details).
4. **Make your changes** and add tests if applicable.
5. **Run the dev server** with `npm run dev` and ensure everything works.
6. **Submit a pull request** with a clear description of your changes.

## Code of Conduct

Please review our [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md) before contributing.

## Issues

If you find a bug or have a feature request, please [open an issue](https://github.com/innacampo/clara/issues).
